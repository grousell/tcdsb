# TCDSB ggplot Theme

TCDSB ggplot Theme

## Usage

``` r
tcdsb_ggplot_theme(text_scale = 1)
```

## Arguments

- text_scale:

  Numeric value to scale font sizes in plot. Default = 1.

## Value

A ggplot theme
