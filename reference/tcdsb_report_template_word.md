# Report Template - Word A function that downloads an .Rmd template, .R script and Word templates for a TCDSB themed report template.

Report Template - Word A function that downloads an .Rmd template, .R
script and Word templates for a TCDSB themed report template.

## Usage

``` r
tcdsb_report_template_word(report_name)
```

## Arguments

- report_name:

  Name of the report to be created.

## Value

Templates and scripts to create the report
