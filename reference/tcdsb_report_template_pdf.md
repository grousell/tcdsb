# Report Template - PDF A function that downloads an .Rmd template, images and .tex files for a TCDSB themed report template.

Report Template - PDF A function that downloads an .Rmd template, images
and .tex files for a TCDSB themed report template.

## Usage

``` r
tcdsb_report_template_pdf(report_name)
```

## Arguments

- report_name:

  Name of the report to be created.

## Value

Templates and images for a TCDSB themed report
