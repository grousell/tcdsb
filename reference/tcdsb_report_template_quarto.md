# Report Template - Quarto PDF A function that downloads an .qmd template and associate .typ and .lua files for a TCDSB themed report template.

Report Template - Quarto PDF A function that downloads an .qmd template
and associate .typ and .lua files for a TCDSB themed report template.

## Usage

``` r
tcdsb_report_template_quarto(report_name)
```

## Arguments

- report_name:

  Name of the report to be created.

## Value

Templates and scripts to create the report
