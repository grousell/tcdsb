# TCDSB gt tables

TCDSB gt tables

## Usage

``` r
tcdsb_gt_table(
  df,
  table_font = "Century Gothic",
  fontsize = 1,
  title = "Title",
  subtitle = NA
)
```

## Arguments

- df:

  A cleaned table for report

- table_font:

  Font

- fontsize:

  Font size scale - base is 1

- title:

  Title of table

- subtitle:

  Subtitle of table

## Value

a gt object
